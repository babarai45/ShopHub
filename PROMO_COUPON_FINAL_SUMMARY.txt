WHAT HAPPENS WHEN...
═══════════════════════════════════════════════════════════════════════════

Admin Creates New Coupon:
  → Not visible on home page (is_featured unchecked)

Admin Checks "Is Featured":
  → Banner appears instantly on home page
  → If coupon is within valid dates and active

Admin Unchecks "Is Featured":
  → Banner disappears from home page
  → Even if coupon is still active

Coupon Validity Expires:
  → Banner still shows (is_featured still checked)
  → But countdown shows "Offer Expired"
  → Users can't apply it anymore

User Visits Home Page:
  → Sees banner with countdown
  → Clicks copy code → Code copied
  → Clicks shop button → Goes to products
  → Later applies code at checkout

Countdown Reaches 0:
  → Timer stops
  → Shows "Offer Expired"
  → Banner still visible but offer ended
  → Users can't apply code
